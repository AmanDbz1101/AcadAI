```mermaid
sequenceDiagram
  actor user as Researcher
  participant fe as React Frontend
  participant api as FastAPI Backend
  participant qabot as QA Bot LangGraph

  fe->>api: POST /api/papers/{id}/chat

  api->>qabot: invoke graph state
  qabot->>qdrant: Hybrid search — BGE dense (384-dim) + BM25 sparse
  qdrant-->>qabot: Top 20 results, RRF fused scores
  qabot->>reranker: Rerank top 20 (ms-marco-MiniLM-L-12-v2)
  reranker-->>qabot: Top 5 chunks with section_title, page_start
  qabot->>groq: System prompt + context + conversation history (last 10)
  groq-->>qabot: Structured markdown answer
  qabot-->>api: {message, retrieved_chunks}

  api->>api: Extract sources from retrieved_chunks
  api-->>fe: {message, sources: [{section_title, page_start}]}
  fe-->>user: Render markdown answer + clickable source boxes

  user->>fe: Click source box
  fe->>fe: handleChatSourceClick() — scrollToSection() or jumpToPage()
  fe-->>user: PDF viewer jumps to source location
```

```mermaid
  SW -- THEORETICAL --> TG["Theoretical guide prompt\nfocus: theorems, proofs"]
  SW -- SURVEY --> SG["Survey guide prompt\nfocus: taxonomy, gaps"]

  NOTE1 --> IX1["Phase 2 — Indexing"]

  PROG2 --> NOTE2["QA bot activates ~99s"]

  PEND --> JOIN1{ }
  NOTE2 --> JOIN1
  JOIN1 --> READ[User reads guide and paper]
  READ --> ASK[User asks question in QA bot]

  ASK --> HYB["Hybrid retrieval BGE + BM25"]
  HYB --> RRF["RRF fusion top 20 results"]
  RRF --> RERANK["FlashRank reranking top 5"]
  RERANK --> CTX["Build context with section labels"]
  CTX --> LLM2["Llama 3.3 70B generates answer\nlast 10 messages history"]
  LLM2 --> ANS["Return answer + source sections"]
  ANS --> RENDER["Render markdown answer"]
  RENDER --> SRC["Show clickable source boxes"]

  CLICK -- yes --> JUMP["scrollToSection() or jumpToPage()"]
```
