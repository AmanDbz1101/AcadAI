THREE-PASS METHOD STRUCTURE
=============================

Pass 1: Quick Scan (5-10 min)
├── Step 1: Title & Abstract → Identify problem
├── Step 2: Introduction → Understand motivation
├── Step 3: Section headings & Figures → Paper structure
└── Step 4: Conclusion → Main claims

Pass 2: Method Understanding (20-40 min)
├── Step 1: Method section → Core approach
├── Step 2: Architecture figures → System design
├── Step 3: Experiments → Evaluation setup
└── Step 4: Results & Tables → Performance

Pass 3: Deep Analysis (1-2 hours)
├── Step 1: Equations & Algorithms → Technical details
├── Step 2: Ablation studies → Component importance
└── Step 3: Limitations & Discussion → Critical evaluation

Final Tasks
├── Summary: Write 5-sentence summary from memory
└── Reflection: Answer deep understanding questions


ROUTING LOGIC
==============

if category == "ORIGINAL_RESEARCH" and query is None:
    → Generate Three-Pass Reading Guide
    → Save to output/<document_id>_guide.json
    
elif query is not None:
    → Retriever (optimize query)
    → Q&A (answer with citations)
    
else:
    → Summarizer (category-specific summary)
